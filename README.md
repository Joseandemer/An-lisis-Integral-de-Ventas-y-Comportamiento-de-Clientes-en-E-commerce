# 📊 Análisis de Ventas y Clientes en E-commerce

## 📌 Descripción
Este proyecto analiza el comportamiento de clientes y el rendimiento de ventas en un entorno de comercio electrónico mediante el uso de SQL y Power BI.

Se desarrollaron KPIs clave y visualizaciones interactivas que permiten entender patrones de compra, preferencias de pago, uso de descuentos y niveles de satisfacción del cliente.

## 🛠️ Herramientas utilizadas
- MySQL
- Power BI
- Excel

## 📊 KPIs desarrollados
- Volumen total de compras
- Monto total recaudado
- Clientes que utilizaron descuentos
- Edad promedio de clientes
- Top 5 categorías con mayor volumen de ventas
- Ventas mensuales
- Métodos de pago más utilizados
- Relación entre envío y satisfacción
- Top ubicaciones con mayor recaudación

## 🎯 Objetivo
Generar insights que permitan mejorar la toma de decisiones comerciales y optimizar estrategias de ventas y experiencia del cliente.

## 📈 Resultados
Se identificaron patrones clave en el comportamiento de compra, destacando el impacto de los métodos de pago, las categorías más rentables y la relación entre logística y satisfacción del cliente.
