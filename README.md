# An-lisis-Integral-de-Ventas-y-Comportamiento-de-Clientes-en-E-commerce
Proyecto de Data Analytics enfocado en el análisis del comportamiento de clientes y rendimiento de ventas en un entorno de comercio electrónico.  Se desarrolló un flujo completo de análisis que incluye limpieza de datos, consultas en SQL, definición de KPIs estratégicos y construcción de un dashboard interactivo en Power BI
